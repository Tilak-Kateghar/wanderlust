WanderLust is a full-stack web application inspired by Airbnb, where users can explore travel listings, view locations on a map, and share their own places with others.

## Features
- User authentication (Sign up / Login)
- Create, edit, and delete property listings
- Upload images for listings
- Interactive map showing listing locations
- Review and rating system
- Authorization for owners and reviewers

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS (Embedded JavaScript templates)
- Bootstrap
- Mapbox API
